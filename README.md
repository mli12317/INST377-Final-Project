# Interactive Art Explorer

## Project Description
  Interactive Art Explorer is a web application that allows users to search and browse artwork from the Art Institute of Chicago API. Users can search for artwork by keyword, browse artists, filter artwork classifications, and save favorite artworks to a database.
This project was created to make museum artwork easier to explore in a simple and interactive way.

## Target Browsers
The website is designed mainly for modern desktop browsers.
This application was tested on:
Google Chrome (Desktop)
Microsoft Edge (Desktop)
